# 🎵 Spotify Music Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Spotify Music Analysis Dashboard built in Power BI.

The dashboard helps analyze songs, artists, albums, popularity, release trends, explicit content, and music performance using interactive visualizations and DAX measures.

It provides valuable insights into Spotify's music dataset through multiple report pages.

---

## 🚀 Objectives

- Analyze Spotify songs and artists
- Identify top-performing songs
- Compare album types
- Understand popularity trends
- Explore release patterns
- Build an interactive dashboard using Power BI

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- CSV Dataset

## 📌 Key Insights

✔ 789 Distinct Songs

✔ 342 Artists

✔ Average Song Duration: 3.28 Minutes

✔ Average Popularity: 89.62

✔ Singles contribute more songs than Albums

✔ Song releases increase during the second half of the year
